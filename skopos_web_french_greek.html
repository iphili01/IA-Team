Σκοπός πολύγλωσσων ιστοσελίδων



Μια πολύγλωσση ιστοσελίδα είναι οποιαδήποτε ιστοσελίδα που προσφέρει περιεχόμενο σε περισσότερες από μία γλώσσες. Παραδείγματα των πολυγλωσσικών δικτυακών τόπων θα μπορούσαν να περιλαμβάνουν μια καναδική επιχείρηση με μια αγγλική και μια γαλλική έκδοση της ιστοσελίδας της ταυτόχρονα. Μία πολυ-περιφερειακή ιστοσελίδα είναι αυτή που στοχεύει ρητά χρήστες σε διαφορετικές χώρες απο την χώρα που βρίσκεσαι. Μέσα απο αυτές τις σελίδες οι χρήστες θα μαθαίνουν νέες λέξεις απο γλώσσες που μπορεί να μην έτυχε μέχρι εκείνη την στιγμή να έρθουν σε επικοινωνία!

But de sites Web multilingues



Un site Web multilingue est un site Web qui propose des contenus dans plus d'une langue. Exemples de sites multilingues pourraient inclure une compagnie canadienne avec une anglaise et une version française du site Web simultanément. Un site multi-régionale vise spécifiquement les utilisateurs dans différents pays du pays où vous êtes. Grâce à ces pages utilisateurs apprendre de nouveaux mots de langues qui ne peuvent pas être reçues dans les temps à entrer en contact!
